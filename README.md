### Cześć! 👋

Mamy tu aplikacje webowe zbudowane w oparciu o framework Django, będące przeglądem zrealizowanych projektów programistycznych. Aplikacje mają różnorodne funkcjonalności - od prostych widoków dynamicznych, przez liczniki odwiedzin, aż po zaawansowane systemy CRUD (Bookhome, Event Planner) oraz komunikację asynchroniczną z wykorzystaniem Fetch API.

Kluczowe elementy:

- Backend: Django (Python) z obsługą baz danych i logiki biznesowej.
- Frontend: HTML, CSS oraz czysty JavaScript do interakcji z API.
- Funkcje: Zarządzanie wydarzeniami, biblioteka książek, lista zadań To-Do oraz mechanizmy głosowania i komentarzy.
